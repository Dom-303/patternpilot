# Promotions

Dieser Ordner enthaelt die von Patternpilot erzeugten Promotion-Pakete fuer `eventbear-worker`.

Zweck:

- Intake-Ergebnisse in eine kuratierbare Form bringen
- den Schritt von Queue zu Landkarte, Learnings und Decisions nachvollziehbar machen
- einen reproduzierbaren Zwischenstand haben, bevor kanonische Artefakte veraendert werden

Wichtig:

- `intake/` ist der rohe projektgebundene Eingang
- `promotions/` ist der kuratierbare Uebergang
- erst nach `promote --apply` werden kanonische Dateien aktualisiert
