# Intake-Dossiers

Hier landen die automatisch erzeugten Intake-Dossiers fuer `eventbear-worker`.

Jedes Dossier startet als Arbeitsgrundlage fuer einen externen GitHub-Fund.
Es ist noch keine kuratierte Wahrheit und darf erst nach Review in `repo_landkarte.csv`, `repo_learnings.md` oder `repo_decisions.md` promoted werden.

Ab Stage 3 enthalten die Dossiers zusaetzlich projektgebundene Alignment-Signale fuer den `eventbear-worker`.
