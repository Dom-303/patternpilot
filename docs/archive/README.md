# Archive

Dieser Ordner ist fuer historische oder interne Materialien gedacht, die nicht mehr Teil der aktiven Produktoberflaeche sein sollen.

Wenn alte Planungs- oder Arbeitsdokumente fuer die laufende Produktnutzung keinen Wert mehr haben, werden sie lieber entfernt als im aktiven Doku-Pfad mitgeschleppt.
