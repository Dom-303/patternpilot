# Pattern Extraction Prompt

Verdichte mehrere ähnliche externe Repos oder Produkte zu wiederverwendbaren Mustern.

Nicht gewünscht:
- bloße Einzelrepo-Zusammenfassungen

Gewünscht:
- Musterfamilie
- wiederkehrende Stärken
- wiederkehrende Schwächen
- typische Risiken
- Bedeutung für das Zielprojekt
- möglicher nächster Schritt
