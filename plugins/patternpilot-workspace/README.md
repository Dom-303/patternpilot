# Patternpilot Workspace Plugin

Repo-lokales Plugin-Scaffold fuer die spaetere Patternpilot-Verteilung in Codex-Workspaces.

## Zweck

- Patternpilot als installierbare Workspace-Erweiterung vorbereiten
- lokale Marketplace-Metadaten bereitstellen
- spaetere Hooks, Skills oder App-Bindungen andockbar machen
